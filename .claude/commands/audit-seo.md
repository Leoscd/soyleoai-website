Realizá una auditoría SEO técnica completa de SoyLeoAI.com.

## Pasos a seguir:

1. **Lee los archivos clave**: index.html, vercel.json, y cualquier página HTML existente

2. **Verificá la presencia de:**
   - Tag <title> con keyword principal
   - Meta description (150-160 chars)
   - Meta robots
   - Open Graph (og:title, og:description, og:image, og:url)
   - Twitter Cards
   - Schema Markup JSON-LD (Person, Course, Review/AggregateRating)
   - Canonical URL
   - Lang attribute en <html>
   - Alt text en todas las imágenes

3. **Verificá archivos técnicos:**
   - ¿Existe sitemap.xml? ¿Está actualizado?
   - ¿Existe robots.txt?
   - ¿vercel.json tiene headers de caché?

4. **Generá un reporte** con este formato:
   - ✅ OK: lo que está bien
   - ⚠️ MEJORAR: lo que falta o está incompleto
   - ❌ CRÍTICO: lo que está roto o ausente

5. **Preguntá a Leo** si quiere que apliques los fixes automáticamente o solo reportar.
